<h1> Download node.js from the official website</h1>

<h4>https://nodejs.org/dist/v24.13.0/node-v24.13.0-x64.msi</h4>

 <h3>After successful installation, open vs code and run the following command : </h3>

    npm create vite@latest

<h3>Give the name to the project as "frontend"<br><br> Select React, then Javascript<br><br> Then run the following command:</h3>

    cd frontend
    npm install
    npm install axios react-router-dom react-toastify

<h3>After this run the project using : </h3>

    npm run dev
